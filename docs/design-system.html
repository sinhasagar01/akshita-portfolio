Rules

The conventions every change holds to.

Tokens not hex. Every value maps to a token in the theme block, nothing references a raw value.
Three animation lanes. Motion for component motion, Lenis for smooth scroll, GSAP for pinned and scrubbed sections.
Route scrolls. Never call lenis.scrollTo directly, every programmatic scroll goes through the scrollToTarget helper.
Reduced motion. A fallback everywhere, and the animation end state still applies under reduced motion.
One breakpoint. Header and hero and footer swap to mobile together at the shared breakpoint.
Canonical host. www.akshitas.com, the apex redirects to it, and metadataBase matches.
